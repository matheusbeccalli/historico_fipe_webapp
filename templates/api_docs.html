Introduction

The FIPE Price Tracker API provides programmatic access to historical vehicle price data from Brazil's FIPE (Fundação Instituto de Pesquisas Econômicas) table. Use these endpoints to query price history, compare vehicles, and analyze market trends.

All responses are returned as JSON. Dates are stored in YYYY-MM-DD format. Prices are in Brazilian Real (BRL) — for example, R$ 56.789,00.

Base URL:

https://your-domain.com

Authentication

The API is public and requires no authentication. No API keys are needed. You can call any endpoint directly from curl, Postman, or your application code.

bash # No authentication needed curl https://your-domain.com/api/brands

Rate Limits

All endpoints are rate-limited by IP address. When you exceed the limit, the API returns HTTP 429 Too Many Requests.

EndpointMethodLimit
/api/brandsGET120/min
/api/vehicle-options/<id>GET120/min
/api/monthsGET120/min
/api/default-carGET120/min
/api/pricePOST60/min
/api/compare-vehiclesPOST30/min
/api/economic-indicatorsPOST60/min
/api/depreciation-analysisGET20/min
/healthGET60/min

Error Handling

Errors return a JSON object with an error field describing what went wrong.

json { "error": "Description of what went wrong" }
400Bad Request — missing or invalid parameters
404Not Found — no data matches your query
429Too Many Requests — rate limit exceeded
500Internal Server Error — something went wrong on our end

POST /api/compare-vehicles

Get price history for up to 5 vehicles. Returns time-series data suitable for charting.

30 requests/minute

Parameters

NameTypeInDescription
vehicle_idsarray of integersJSON body requiredList of ModelYear IDs to compare (max 5)
start_datestringJSON body optionalStart date filter (YYYY-MM-DD)
end_datestringJSON body optionalEnd date filter (YYYY-MM-DD)

Request

bash curl -X POST https://your-domain.com/api/compare-vehicles \ -H "Content-Type: application/json" \ -d '{"vehicle_ids": [2100, 2101], "start_date": "2023-01-01", "end_date": "2024-06-01"}'
Response 200 OK
json { "vehicles": [ { "name": "Volkswagen Gol 1.0 12V MCV 2024 Gasolina", "year_id": 2100, "data": [ { "date": "2023-01-01", "price": 55230.00, "label": "janeiro/2023" }, { "date": "2023-02-01", "price": 55890.00, "label": "fevereiro/2023" } ] } ] }

Errors

400Missing vehicle_ids or exceeds maximum of 5
404No price data found for the given vehicles
429Rate limit exceeded
500Database query failed
POST /api/price

Look up a single price point for a specific vehicle at a specific reference month. Uses fuzzy matching for brand and model names.

60 requests/minute

Parameters

NameTypeInDescription
brandstringJSON body requiredBrand name (fuzzy match, e.g. "Volkswagen")
modelstringJSON body requiredModel name (fuzzy match, e.g. "Gol")
yearstringJSON body requiredYear description (exact match, e.g. "2024 Gasolina")
monthstringJSON body requiredReference month date (YYYY-MM-DD, e.g. "2024-01-01")

Request

bash curl -X POST https://your-domain.com/api/price \ -H "Content-Type: application/json" \ -d '{"brand": "Volkswagen", "model": "Gol", "year": "2024 Gasolina", "month": "2024-01-01"}'
Response 200 OK
json { "brand": "Volkswagen", "model": "Gol 1.0 12V MCV", "year": "2024 Gasolina", "month": "janeiro/2024", "month_date": "2024-01-01", "price": 56789.00, "price_formatted": "R$ 56.789,00", "fipe_code": "026011-6" }

Errors

400Missing required parameters (brand, model, year, month)
404No price found for the given combination
429Rate limit exceeded
500Database query failed
